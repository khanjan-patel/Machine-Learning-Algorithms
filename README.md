# Machine Learning Algorithms

This repository contains implementations of fundamental **Machine Learning algorithms** using **Python** and **Scikit-learn**. Each algorithm is organized in a separate folder with well-structured code, data preprocessing, model training, evaluation, and visualization where applicable.


1. Clone the repository.
2. Install the required libraries.
3. Open the desired algorithm folder.
4. Run the Python file.

This repository was created to practice and understand core Machine Learning algorithms, data preprocessing, model evaluation, hyperparameter tuning, clustering techniques, and dimensionality reduction.
